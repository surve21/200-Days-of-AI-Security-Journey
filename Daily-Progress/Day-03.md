##  Networking

### Module 5: Network Topologies

**Topics Covered**

* Bus Topology
* Star Topology
* Ring Topology
* Mesh Topology
* Tree Topology

**Key Learning**
Each topology has its own advantages, disadvantages, scalability, and fault tolerance characteristics. Choosing the right topology depends on network size, performance requirements, and reliability.

---

### Module 6: Routing and Switching

**Topics Covered**

* Introduction to Routing
* Introduction to Switching
* Differences between Routers and Switches
* Packet Forwarding
* MAC Address vs IP Address
* Role of Routing and Switching in Enterprise Networks

**Key Learning**
Switches forward frames within a Local Area Network (LAN) using MAC addresses, while routers connect different networks and forward packets using IP addresses.

---

##  Cisco Packet Tracer Practice

### Experiment 5: Spanning Tree Protocol (STP)

**Objectives**

* Understand the purpose of STP.
* Observe how STP prevents switching loops.
* Identify the Root Bridge.
* Analyze blocked and forwarding ports.

**Outcome**
Learned how Spanning Tree Protocol prevents broadcast storms and network loops by creating a loop-free logical topology while maintaining redundant links for fault tolerance.

---

##  Key Takeaways

* Network topology influences performance, scalability, and reliability.
* Switching enables communication within the same network, while routing enables communication between different networks.
* Spanning Tree Protocol (STP) is essential for preventing Layer 2 switching loops and ensuring stable network operation.
